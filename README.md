# Student-Manage-System

Studnent manage system that be powered by C++ have 6 actions is a simple Program.

- search student info
- add student info
- del student info
- search student point
- add student point
- del student point

## Used

### Environment

- Windows 10 64bit
- Visual Studio 2017

### Install

Download and use `code.cpp` in Visual Studio 2017.
